# FFmpegVideoRange2

## 此项目是android 仿照剪映一些功能，暂时只有预览条的精准快速抽帧显示，以及滑动过程中与播放逐帧显示，我观察了下，基本只有大厂的剪辑软件中会有这样的功能。
## 网上的一些开源项目中，都仅仅是非精准抽帧或者只是关键帧，有些速度很慢。

## 本项目抽帧速度已经和快手的快影差不多了，但是比剪映还是要慢一些，大家可以实际对比一下。
